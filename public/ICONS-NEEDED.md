# App Icons Needed

To make the app installable, you need these icon files in the `/public` folder:

## Required Files:
1. **icon-192.png** - 192x192 pixels
2. **icon-512.png** - 512x512 pixels

## How to Create Icons:

### Option 1: Use an Online Tool
1. Go to https://realfavicongenerator.net/
2. Upload your logo (should be square, min 512x512)
3. Download the generated icons
4. Rename them to match the filenames above
5. Place in `/public` folder

### Option 2: Use Canva
1. Create a 512x512 design with your logo
2. Export as PNG
3. Use an image resizer to create 192x192 version

### Option 3: Simple Purple Circle (Placeholder)
- Background: #a855f7 (purple)
- Text: "OSHO" in white
- Save as 512x512 PNG, then resize to 192x192
